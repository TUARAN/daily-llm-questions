# Tools

> 将重复的 LLM 交互流程化、自动化

## 工具列表

### ask.py

简单的命令行工具，用于快速向 LLM 提问并记录结果。

**使用方式：**
```bash
python ask.py "你的问题"
```

**功能：**
- 快速提问
- 自动记录到 questions/ 目录
- 支持多模型切换

## 使用建议

1. **从简单开始**
   - 不要一开始就追求完美的工具
   - 先用手工流程，发现痛点
   - 再针对性地开发工具

2. **保持轻量**
   - 工具应该简单易用
   - 减少配置和依赖
   - 专注核心功能

3. **持续迭代**
   - 根据实际使用调整
   - 记录改进想法
   - 定期重构优化
