# Thinking Prompts

## 深度思考模板

```
我想深入思考一个问题：[问题]

请帮我：
1. 拆解这个问题的核心要素
2. 识别可能的隐含假设
3. 提供多个不同角度的分析框架
4. 指出我可能忽略的方面

不要直接给答案，而是引导我思考。
```

## 概念澄清

```
我对 [概念] 的理解是：[我的理解]

请帮我：
1. 验证这个理解是否准确
2. 补充我遗漏的关键点
3. 用类比或例子加深理解
4. 指出常见的误解
```

## 决策辅助

```
我面临一个决策：[决策内容]

已知条件：
- [条件1]
- [条件2]

请帮我：
1. 列出需要考虑的关键因素
2. 分析每个选项的利弊
3. 识别潜在的风险和机会
4. 提供决策框架而非直接建议
```

## 使用技巧

- 这些模板是起点，根据实际情况调整
- 重点是引导思考，而非获取答案
- 记录使用后的效果和改进
