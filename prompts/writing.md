# Writing Prompts

## 文章大纲生成

```
我想写一篇关于 [主题] 的文章。

目标读者：[读者画像]
核心观点：[核心观点]
期望长度：[字数]

请帮我：
1. 设计文章结构大纲
2. 为每个部分提供关键点
3. 建议开头和结尾的方式
4. 指出可能需要的例子或数据
```

## 改写优化

```
原文：
[粘贴原文]

优化目标：
- [ ] 更简洁
- [ ] 更有说服力
- [ ] 更通俗易懂
- [ ] 更符合[风格]

请提供改写建议，并说明理由。
```

## 标题生成

```
文章内容：[简要描述或粘贴正文]

请生成10个候选标题，风格包括：
- 直接型（清晰表达主题）
- 疑问型（引发思考）
- 对比型（制造反差）
- 数字型（具体可感）
- 故事型（引入场景）

每个标题都说明适用场景。
```

## 文章润色

```
请帮我润色这段文字：

[原文]

保持原意，优化：
1. 句式节奏
2. 用词精准度
3. 逻辑连贯性
4. 可读性

标注改动的地方并说明理由。
```

## 使用经验

- 先有自己的思路，再用 Prompt 辅助
- 不要完全依赖生成内容，要保持个人风格
- 多次迭代，逐步优化
